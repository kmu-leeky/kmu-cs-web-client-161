  1. What does HTML stand for? (1 point)

    • T : text
    • M : markup
    • L : language
  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)
    3. Answer: programming language 는 컴퓨터 시스템을 구동시키는 소프트 웨어를 작성하기 위한 언어이고, markup language는 데이터를 기술하기 위해 사용되는 언어이다.
  3. Describe what do the following tags mean(4 point)
    • pre:태그 사이에 입력된 대로 출력이 된다.
    • p: 별도의 단락을 나타낸다. 태그가 끝나면 줄바꿈이 일어난다.
    • br:끝나는 태그 없이 단독으로 쓰이며 다음줄로 넘어가는 enter와 같은 역할을 수행한다.
    • h1, h2, h3, h4, h5, h6: 정보의 계층을 구조화 해준다.
  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer: strong,em 은 의미적으로 강조되어 각각 굵게, 이테릭 체를 나타낸다. b,i는 단순히 보기에만 강조되어 굵게, 이테릭 체로 나타난다. 시각장애인 과 같은 사람들이 볼때에는 strong,em 이 강조되어 나타난다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: alt는 이미지가 표시되지 않거나 표시하지 않을때 이미지를 대신하여 나타나는 텍스트이고, title은 이미지의 제목을 나타낸다.
  6. In table, describe difference between th and td (5 point). Answer: th는 표의 제목을 나타낼때 쓰인다.기본적으로 가운데 정렬이 되어서 나타난다. td는 셀을 만들때 쓰인다. 왼쪽정렬이 되어서 나타난다.
