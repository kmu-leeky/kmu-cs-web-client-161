  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)
    3. Answer: Markup language is designed to format text and Programming language is an set of instructions to the computer to perform. The difference is that Markup language is interpreted by the browser but Programming language is compiled.
  3. Describe what do the following tags mean(4 point)
    • pre: represents a block of preformatted text/is displayed in a fixed-width font and preserves both spaces and line breaks
    • p: separates "paragraph"
    • br: insert a new line
    • h1, h2, h3, h4, h5, h6: organizing contents hierarchically / the font size: h1 > h2 > h3 > h4 > h5 > h6
  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer: The former things are implication/semantic and the latter things are simply styling.It means that if things like screenreader read contents interpreting meaning, the 'strong' and 'em' function as right.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: alt tag is a message that will be shown when the image is not available and it differs from title in that title always has explanation.
  6. In table, describe difference between th and td (5 point). Answer: 'th' implies title of each column and 'td' is an element.
