  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)
    3. Answer: Markup Language에서는 연산이 안되고, 구조적인 작성으로 인해 텍스트만 출력해주는 언어이다. 그리고 컴파일이 되는게 아니라 그냥 브라우저에 의해 읽히기만 하는 것이다.
    Programming Language에서는 연산이 가능하고, 구조적인 작성으로 알고리즘을 구현시켜주는 언어이다. 컴파일이 실행되며 실행파일을 만들어준다.
  3. Describe what do the following tags mean(4 point)
    • pre: 메모장에 입력한 그대로(예를들면 줄바꾸기, 들여쓰기 그대로) 출력이 가능한 tag
    • p: p tag안에 들어있는 문장을 한 문장으로 생각한다.
    • br: 다음 문장으로 넘겨주는 tag, br태그는 쌍을 이루지 않는다.
    • h1, h2, h3, h4, h5, h6: 목차의 의미를 두는 tag, 글자의 크기를 정의할 때 쓰는 것이 아니다.
  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer: strong, em태그는 글자에 강조를 주기위해(browser마다 강조형태가 다름)쓰는 것이고, b, i태그는 단순히 굵게, 기울이게로 글자에 효과를 주는 tag이다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: alt태그는 사진이 정상적으로 출력 되지 않을 때 나타나는 문장을 담는 곳이고, title태그는 사진에 마우스를 올렸을 때 말풍선 형식으로 출력되는 문장을 담는 곳이다.
  6. In table, describe difference between th and td (5 point). Answer: th태그는 단어를 가운데정렬과 굵게해서 표 안에 넣어주는 tag이고, td태그는 세로(row) 형식으로 글을 표안에 넣어주는 tag이다.
