  3. Describe what do the following tags mean(4 point)
    • pre: 엔터,스페이스 등을 입력하였을 때,별도의 태그 사용 없이 입력 그대로 보여주는 태그

  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: alt 태그는 그림이 로딩되지 않았을 경우, 그림 대신에 설명 문구가 나타나는 태그이다. title은 그림 위에 마우스를 갖다대면 설명이 나오는 태그이다.
  6. In table, describe difference between th and td (5 point). Answer: table 에서 th는 제목, td는 내용을 나타낼 때 쓰는 태그이다.
