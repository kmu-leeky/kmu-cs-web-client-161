  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)

  3. Describe what do the following tags mean(4 point)
    • pre: 입력하는 그대로 보여준다. 다만 안에 다른 태그를 사용할 수는 없다.
    • p: Separates Paragraph. 분리된 단락.
    • br: Insert a new line. 개행.
    • h1, h2, h3, h4, h5, h6: 겉보기에는 글자 크기만 달라보이나 사실은 heading 이라하여 각 항목의 우선순위를 구별하는데 쓰는것이 옳다. 위키백과를 생각해 봅시다.
  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer: strong 이나 em 태그는 의미를 부여하는 것이다. b,i와 겉보기의 형태는 같으니 시각 장애인용 프로그램을 사용하면 strong이나 em은 엑센트를 다르게 하여 출력한다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: title은 이미지가 출력되었을때 마우스 포인터를 올리면 팝업으로 뜨는 메세지. alt는 이미지 출력 실패시(링크 끊김 등등) 파일명대신 나오는 메세지이다.
  6. In table, describe difference between th and td (5 point). Answer: th는 내용에 굵은 글자와 가운데 정렬이 된다. 큰 항목에다 사용하면 좋다. td는 일상적인 얇은 폰트로 나오고 자동으로 가운데 정렬이 되지는 않는다.
