  2. Describe difference between Markup language(HTML) and Programming language, such as Java (10 point)
    3. Answer: Markup언어는 구조적 데이터를 표현해주는 언어이다. HTML이 대표적인 markup언어인데, 단순한 텍스트 문서랑 비교하면 문서의 일부분이 제목이거나 다른부분이 리스트인 것을 명확히 할 수 있게 해준다. Programming언어는 데이터를 변형시켜서 사용되는 언어이다. 입력 데이터를 출력 데이터로 다시 써주는 CPU 인스트럭션을 만들어줌으로써 작동한다.
  3. Describe what do the following tags mean(4 point)
    • pre : 텍스트를 그대로 보여주는 태그, 공백 혹은 개행이 그대로 출력된다.
    • p : 문단을 구분할 때 사용한다.
    • br : 새로운 한 줄을 삽입할 때 사용한다.
    • h1, h2, h3, h4, h5, h6 : 제목을 작성하는 태그, h 뒤에 숫자가 커질수록 작은 제목을 표시한다.
  4. Describe diffrence between using strong, em tag and b, i tag (5 point).
    Answer : strong, em은 의미부여를 위함이고 b,i는 간단한 스타일링을 하기 위함이다. b,i는 앞이 보이지 않는 사람들에게는 효과가 없다. 참고로, strong은 볼드체로 쓰는 것이고, em은 이탈릭체로 쓰는 것이다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point).
    Answer : alt는 이미지가 이용될 수 없을 때 보여질 메세지이다. title은 이미지의 제목이다. alt는 이미지의 의미를 알려주는 역할이므로 title과는 다르다.
  6. In table, describe difference between th and td (5 point).
    Answer : th는 table head의 약자로, 표의 제목을 쓰는 역할이고 굵은 글씨체에 중앙정렬이 된다. td는 table data의 약자로, 셀을 만드는 역할이고 보통 글씨체에 왼쪽 정렬이 된다.
